Experimento Aleatorio

Un experimento aleatorio es aquel cuyo resultado no se puede predecir.

Ejemplo: lanzar un dado

Espacio muestral = {1,2,3,4,5,6}

Probabilidad Simple

Probabilidad Compuesta

Ocurre cuando se analizan dos o más eventos.

Ejemplo: dos monedas → P(cara y cara) = 0.25

Variable Aleatoria

Una variable aleatoria es una función que asigna un valor numérico a cada resultado de un experimento aleatorio.

Definición formal: X : E → ℝ

Tipos

  • Discreta: toma valores contables (ej. dado).
  • Continua: toma infinitos valores en un intervalo (ej. tiempo).

Ejercicio 1

Se lanza un dado una vez. La variable aleatoria X representa el número obtenido.

Ejercicio 2

Se mide el tiempo que tarda una persona en correr 100 metros. La variable X representa el tiempo en segundos.
